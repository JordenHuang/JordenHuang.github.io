[Hexo] 建部落格歷程-主題篇

發布於 2023-08-31
Blog>Hexo
1k words
  • Hexo
  • theme vivia

前言

在架設部落格時，選擇一個喜歡的主題來當作自己的門面很重要。我在選擇主題時，瀏覽了好多Hexo的主題，但一直沒找到滿意的 (對😅，三百多個主題還找不到個滿意的)

就在我以為只能自己動手做時，終於發現了這個 Vivia主題
這主題對我來說真的是一見鍾情，雖然還是有想自己動手更改的地方。於是我就馬上下載，開始使用~

下載與配置

我下載時的version是0.4.0

照著vivia主題文件的指示使用npm下載，然後到node_modules/hexo-theme-vivia/把example_config.vivia.yml複製到根目錄下並改名為_config.vivia.yml
(我自己是使用git安裝然後放在themes資料夾內，一樣要把config檔案複製、移動並重命名)

然後到根目錄把config.yml裡的theme改成vivia

動手時間

雖然找到了滿意的主題，但再經過一些修改，還可以變得更滿意(?)。以下來記錄一下我修改了哪些地方:

1 移除側邊欄的archive跟recent_posts

把sidebar裡的archive跟recent_posts註解，因為不想顯示在側邊欄裡

_config.vivia.yml

2 加強陰影顏色，並調整陰影成偏紫色

source/css/style.styl

3 將原本的文章預覽更改成自己寫在fontmatter中的description

layout/_partial/article.ejs

4 新增顯示更新日期的區塊

layout/_partial/post/date.ejs

5 back to top botton

將back-to-top-btn的樣式額外寫，使他不會在螢幕較小的裝置消失
將back-to-top-wrapper的position改成fixed，z-index增加，以及調整back-to-top-btn距離bottom的位置

source/css/_partial/back-to-top.styl

6 code block的顏色

把code block的顏色調亮一點

source/css/sytle.styl

7 文章資訊

因為前面自己增加了更新日期的區塊，導致在螢幕較小的裝置上會跑出螢幕外，所以要增加x方向的scrollbar

8 顯示的語言

修改zh-TW，因為想要某些地方是英文顯示，例如字數的地方顯示words而非詞

未來還想更改的地方

  • 點選分類或標籤時，不是顯示archive，而是相對應的分類或標籤(Ex: ‘Blog/‘ or ‘Blog/Hexo/‘)
  • Scrollbar的樣式
  • code block(程式碼區塊)增加複製按鈕(也許參考nexmoe主題)以及展開收合按鈕
  • 修改404頁面的樣式

文末

以上就是本篇的全部內容啦，因為筆者我才剛開始學有關網頁的程式，還有許多東西不懂也不知道如何修改，等到之後學多一點知識之後，如果有再對這主題的個人修改，會更新在本篇或另寫一篇來記錄(也不排除做出自己的主題啦😂)。
